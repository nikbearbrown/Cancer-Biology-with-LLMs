#### Advanced Verification Exercises

**Exercise 10.1: Mechanism Verification**
- Claim: "BCL-2 overexpression blocks mitochondrial pathway"
- Task: Build evidence map from 5+ papers
- Challenge: Papers describe mechanism differently - synthesize
- Deliverable: Your own mechanistic diagram with sources

**Exercise 10.2: Therapeutic Claims**
- Claim: "BH3 mimetics show promise in treating [X] cancer"
- Task: Find clinical trial data (ClinicalTrials.gov + papers)
- Analysis: "Promise" is vague - quantify effectiveness
- Critical thinking: Marketing language vs. scientific evidence

**Exercise 10.3: Context Matters**
- Topic: Apoptosis resistance mechanisms
- Task: Find cases where standard mechanism doesn't explain resistance
- Synthesis: What's missing from simplified models?
- Meta-lesson: Biology is messier than textbook diagrams

#### Investigation Sidebar: Where LLMs Struggled
- Complex pathway interactions: High disagreement between models
- Context-dependent activation: Required 10+ sources to clarify
- Therapeutic efficacy: LLMs cited outdated trials - manual verification needed
